This is my final project. The name of the program is "Trace and Learn".
To use this program simply click on the "Letters" button or the "Numbers" button.
Once you have done that, you can choose any of the listed letters and numbers you
would like to trace. Now that you have chosen a character to trace, you can
select from the color buttons and click and hold the mouse button while
moving the mouse across the character to trace it. After that you can
return to the start screen with the "Home" button or use the "Clear" button to retrace your current character!
